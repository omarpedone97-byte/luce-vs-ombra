# Luce vs Ombra — Setup su Glitch

## 1. Carica il progetto
1. Vai su glitch.com e accedi
2. "New Project" → "Import from GitHub" oppure crea un progetto Node vuoto
3. Se crei un progetto vuoto: apri ogni file di questa cartella e incolla il contenuto nel file corrispondente su Glitch (crea `public/index.html`, `server.js`, `package.json` con gli stessi nomi)
4. Se Glitch supporta l'importazione da zip: trascina il file .zip direttamente nell'interfaccia

## 2. Imposta il tuo username TikTok (SENZA la @)
1. Nel pannello Glitch, apri il file `.env` (se non esiste, crealo)
2. Scrivi dentro: `TIKTOK_USERNAME=iltuousername`
3. Salva — Glitch riavvia da solo il progetto

## 3. Prendi il link dell'overlay
1. In alto a sinistra su Glitch, clicca "Share" → copia il link tipo `https://nome-progetto.glitch.me`
2. Questo è il link da incollare in TikTok LIVE Studio come sorgente "Link"

## 4. Come funziona l'assegnazione squadra
- Gli spettatori scrivono "LUCE" o "OMBRA" in chat UNA VOLTA
- Da quel momento, ogni loro regalo conta per quella squadra
- Se non scelgono mai, vengono assegnati a caso (e ricordati per tutta la live)

## 5. IMPORTANTE — Glitch va in stand-by
Glitch "addormenta" i progetti gratuiti dopo 5 minuti senza traffico.
Prima di andare live, apri il link dell'overlay nel browser un minuto prima
per "svegliarlo", oppure usa un servizio gratuito come UptimeRobot per
tenerlo sveglio automaticamente durante le tue dirette.

## 6. Test prima di andare live
Il pannello di test (pulsanti 1-6 per livello) funziona ANCHE quando sei
collegato al server — puoi continuare a usarlo per provare gli effetti.
Premi "T" per nasconderlo prima di andare live davvero.
